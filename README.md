# MAX REMY DEV - PORTFOLIO

[maxremy.dev](https://maxremy.dev/) built with :

## FRONTEND

-   **[Next.js (App Router)](https://nextjs.org/)** – React.js framework with SSR, SSG, and nested layouts
-   **[TypeScript](https://www.typescriptlang.org/)** – Static typing for better reliability and clarity
-   **[Tailwind CSS](https://tailwindcss.com/)** – Utility-first CSS framework for rapid and responsive design
-   **[Shadcn/UI](https://ui.shadcn.com/)** – Modern and accessible component library
-   **[Framer Motion](https://motion.dev/)** – Smooth and dynamic UI animations
-   **[AWS Amplify](https://aws.amazon.com/amplify/)** – Frontend hosting with continuous deployment

## BACKEND

-   **[Strapi CMS](https://github.com/BabylooPro/NEW-PORTFOLIO/tree/main/cms)** – Self-hosted headless CMS for content management
-   **[AWS EC2 (Linux)](https://aws.amazon.com/ec2/)** – Hosting server for the backend and Strapi
-   **[MySQL](https://www.mysql.com/)** – Relational database for Strapi content
-   **[AWS S3](https://aws.amazon.com/s3/) + [CloudFront](https://aws.amazon.com/cloudfront/)** – Scalable media storage and delivery via CDN
-   **[ContactForm.csharp](https://github.com/BabylooPro/ContactForm.csharp)** – A simple home-made API to add a contact form to any website (email sending, templates, attachments)

## FEATURES

-   **Dynamic Routing**: Built with Next.js App Router for SSR, SSG, and nested layouts
-   **Modern UI**: Minimalist and clean design powered by Shadcn/UI and Tailwind CSS
-   **Theme Support**: Supports dark, light, and system themes
-   **Fully Responsive**: Optimized for mobile, tablet, and desktop experiences
-   **[Framer Motion Animations](https://motion.dev/)**: Smooth transitions and animations for a polished UX
-   **[GitHub Integration](https://docs.github.com/en/graphql)**: Automatically fetches and displays pinned and top-starred repositories using the GitHub GraphQL API
-   **[Strapi CMS](https://strapi.io/)**: Self-hosted CMS on EC2 for managing dynamic content
-   **Media Delivery**: Assets served via AWS S3 and CloudFront for performance and scalability
-   **[CI/CD Workflow](.github/workflows)**: GitHub Actions for containerization, build, and deploy
-   **Preview Deployments**: A preview branch and environment are used to test code before merging to main website production
